17 *
18 * This file defines the Output class in the orange\framework namespace.
19 * It implements the OutputInterface and extends the Singleton base,
20 * meaning there is only ever one instance used during the application lifecycle.
21 * Its role is to manage all HTTP output — headers, status codes, content type, charset, buffering, redirects, and sending responses.
22 *
23 * ⸻
24 *
25 * 1. Core Responsibilities
26 *  •   Response management: keeps track of what content and headers should be sent.
27 *  •   Headers and status codes: allows setting, replacing, and flushing HTTP headers, as well as configuring response codes.

53 *  •   get() → retrieves current buffer content.
